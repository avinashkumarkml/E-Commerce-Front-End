### <u>Run Locally</u>

Clone the project

```
git clone https://github.com/shuklabhisekh/Backened_Of_Theory_Clone.git
```

Install npm packages

```
npm install
```

Run the surver

```
npm run server
```

View in Browser

```
go to http://localhost:5000
```
